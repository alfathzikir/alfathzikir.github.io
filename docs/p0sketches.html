Interview with Client

Notes jotted down during the interview with Austin. Common themes have been circled. Coming up with ideas that incorporated a sandbox environment and self-directed actions were difficult when considering the limiting factor of having a single button be the only source of interaction.

Idea: Emotion Button

A wearable that displays the user's personal depiction of the current self

Idea: Randomized Telescope

Pressing the button whirls the telescope around, giving the user a different view of the sky each time

Idea: One-Sound-Recorder

Press the button to record one sound. A random song is generated using that sound

Idea: Warp

Button warps player to different locations in space

Six Gamey Ideas

1. Rhythm game: taps, slides, holds, and releases

2. Contextual button: changes functionality based on location

3. Puzzle game: place and use the button in the right place and order to advance

4. Sidescrolling shooter: shoot things

5. Kinetic visual novel: select a scene and advance

6. Timing game: hold button to fully charge, see how fly you can send rocket

Idea: Tower Defense Game

Place turrets on planets to defend against incoming enemies

Detailed Idea: NightSky

Use stars to generate quotes about the "self"
