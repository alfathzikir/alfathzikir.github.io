The Watermelon War is a simple animation using my sketches!

The Watermelon War is the result of trying to find the best way a single click of button can do to delight my client, Bader Abdulwaseem. After having interviewed him, his passion of creating mods for games such as GTA5 and Garry's Mod struck me. He is currently trying to create a watermelon cannon mod for Garry's Mod to start a watermelon war. But I am here, instead, to bring the watermelon war to him.





The main component of this project is creating the sketching in order to create the animation.

Below you could see the prototypes of character design that I have done.




With all the character design done, then I had to start learning how to sketch watermelon, and broken watermelon.




With all the components done, all that was left was to create a video and implement a button to play that video.

LET THE WATERMELON WAR BEGIN!
