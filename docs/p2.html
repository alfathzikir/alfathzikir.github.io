The Relationship 

CODE   LIVE



The Relationship is a project that is intended to create something in order to have more interactivity through video calling.

The main client for this project is Rhys. He is currently 8 months old and loves to talk with his Oma and Opa. Unfortunately, due to the current COVID situation, all the talking is done through the video calling. This projects tries to add more interactivity betwee Oma/Opa and Rhys during video calls.



There are two main buttons for Oma and Opa to interact with Rhys: Get Attention and Put Bubbles.


With the "Get Attention" button, Oma and Opa can try and get the attention of Rhys. This does so by playing a rattling sound and changing the background.

With the "Put Bubbles" button, Oma and Opa can put bubbles on Rhys' screen which Rhys can then pop.



With these two simple buttons, I believe Oma and Opa can have more interactivity when video calling Opa!
